Parse-server Backend As A Service

Parse-server is an open source version of the Parse backend that can be deployed to any infrastructure that can run Node.js. Parse Server works with the Express web application framework. It can be added to existing web applications, or run by itself.

Template engine

This repository use swig, a simple, powerful, and extendable JavaScript Template Engine.

Database

This repository use mongoose ODM, an elegant mongodb object modeling for node.js

MongoRestController

To enable rapid REST API prototyping you can use microscope mongo-rest-controller NPM package. 

npm i mongo-rest-controller --save
	
import MongoRestController from 'mongo-rest-controller';
import Article from '../models/Article';

export default class ArticleController extends MongoRestController {
    get baseUrl(){
        return '/api/articles'
    }

    get model(){
        return Article;
    }
}
